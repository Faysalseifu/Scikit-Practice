# Scikit-Learn Practice

This repository is a lightweight workspace for practicing classic machine learning with scikit-learn.

## What’s Included

- `notebooks/` for guided practice by topic
- `src/` for reusable Python code and helpers
- `data/raw/` and `data/processed/` for datasets
- `models/` for saved estimators
- `reports/figures/` for generated plots

## Starter Notebooks

- `notebooks/00_start_here.ipynb`
- `notebooks/01_regression.ipynb`
- `notebooks/02_classification.ipynb`
- `notebooks/03_clustering.ipynb`
- `notebooks/04_preprocessing.ipynb`
- `notebooks/05_model_evaluation.ipynb`

## Entry File

- `src/data/make_dataset.py` is a simple starting point for data-loading scripts.

pip install -r requirements.txt
